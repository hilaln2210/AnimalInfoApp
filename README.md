# AnimalInfoApp

אפליקציית אנדרואיד להצגת מידע על בעלי חיים.

## טכנולוגיות
- **Kotlin** · **Android** · **Gradle**

## הרצה
```bash
./gradlew assembleDebug
# או ל-Android Studio: פתחי את הפרויקט והרצי
```

## מבנה
```
app/src/main/
├── java/          # קוד Kotlin
├── res/           # משאבים (layouts, strings, drawables)
└── AndroidManifest.xml
```
